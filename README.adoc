== JBake

=== 本地测试
执行以下命令，并启动本地Jetty服务器，方便在浏览器中预览。
[source,bash]
----
jbake -b -s
----

但是发布的时候不要 `-s` 选项， 否则生成的feed.xml文件中的链接地址就会变成 `http://localhost:8820/`。
